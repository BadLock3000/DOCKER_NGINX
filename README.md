Данный репозиторий отвечает за запуск проекта, а также его сборку.
структура файлов:
PROJ:
*-.env(file)
*-docker-compose.yml(file)
*-nginx(file)
*-front(FOLDER)
*-back(FOLDER)

Нужно создать единую папку, в нее загрузить docker-compose,nginx,.env,front и back
